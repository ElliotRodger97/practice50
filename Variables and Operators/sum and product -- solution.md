# Sum and Product

## Possible Solution

```c
#include <cs50.h>
#include <stdio.h>

int main(void)
{
    // declare and set variables    
    int x = 10;
    int y = 23;
    
    // print variables
    printf("The sum of the two variables is: %d \n", x + y);
    printf("The product of the two variables is: %d \n", x * y);
}
```
